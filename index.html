Feb 25, 2012

I decided to start exploring the world of mobile development. Since I have an android phone (and know how to code in java), an android application seemed like the best place to get started. My next couple blog entries, starting with this one, will detail the steps needed to get an idea onto your phone.


First, I settled on an easy-to-get-off-the-ground java application: a deck of cards. It's really easy to write and test, and it should be easy for me to add in the libraries necessary to port it to my phone.


Here's the code that I wrote:

public enum Suit { CLUBS, DIAMONDS, HEARTS, SPADES, } public enum Rank { TWO, THREE, ... KING, ACE, } public class Card { private final Suit suit; private final Rank rank; public Card(Suit suit, Rank rank) { this.suit = suit; this.rank = rank; } @Override public String toString() { return rank.toString() + suit.toString(); } } public class Deck { private final List cards; private final Random rng; public Deck() { this.rng = new Random(); cards = new ArrayList(); for (Suit suit : Suit.values()) { for (Rank rank : Rank.values()) { cards.add(new Card(suit, rank)); } } } public Card getCard() { int pos = rng.nextInt(cards.size()); return cards.remove(pos); } public boolean hasCards() { return !cards.isEmpty(); } public static void main(String[] args) { Deck deck = new Deck(); while (deck.hasCards()) { Card card = deck.getCard(); System.out.println(card); } } }

Coffee Table 100%
     My roommate and I came across an old coffee table at a garage sale a while back. It was stained a light yellowish color, but suffered from some large water marks. We bought it for $20 with the intention of fixing it up. We detached the top from the base and treated it with a rich, dark, resin-colored wood stain and a triple-coat of polyethylene finish. A few years later (ugh, I know), I finally decided to put an end to this project by buying a bunch of pipe fittings and building legs. It turned out a lot like the table shown in this blog. I ended up using galvanized pipes instead of black ones (too much sticky gunk on the black ones), but otherwise was just as easy as advertised. I'm really happy with how it turned out.

Sept 2012
Irish Stout

For my third brew, I decided to try making an "Irish Stout." Like my previous two, this batch is from an ingredient kit that I purchased on Amazon from a company called "The Home Brewery." I was quite pleased with how "Wheat Beer" and "German Oktoberfest" turned out, so I have high hopes for this one as well. Given that it's all in one kit, it's pretty much just following directions. That being said, I felt like I was much more in tune with what was going on at each step of the process and what to look out for along the way. I think after one or two more of these easy brews, I'll move up to more complicated ones. (I'd really love to give Obama's White House beer a try once I get a secondary fermenter).


Things of note that were different from my previous brews:

  • Specialty grain steepage step had specific temperature requirements. I decided that while I was out at Safeway getting 5 gallons of distilled water (super cheap and convenient for both sterilization and measurement), that I would get a thermometer. I was very glad I did, as it was clear that different colors were going into the wort as the temperature got hotter. At around ~150F, very aromatic smoke/steam started coming off the pot too. Given the harsh warnings about going over 170F, I'm glad I was able to get the most out of the grains without adding bad tastes to the beer.
  • This beer had more ingredients than my previous two brews. I was adding things to the wort pre-boil, 45 minutes in, 50 minutes in, and 58 minutes in.
  • The instructions of this beer indicated that I should bring the temperature of the wort down to room temperature as soon as possible. I did a pseudo ice-bath in my sink, but otherwise didn't know what technique I was supposed to use. I guess if the beer turns out bad I should investigate alternative options to properly chill the wort.


Hopefully I followed directions ok, didn't introduce bacteria, and didn't kill the yeast. This one should be ready right as the first pitches of the new wild-card games are thrown in early October. Cheers!
